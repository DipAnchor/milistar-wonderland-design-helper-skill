# 千星奇域 · 已知能力边界

> 本文件记录千星奇域编辑器的已知能力边界和限制。
> 不包含实现细节，仅标记可行性状态。

## 分类标记
- ✅ 已知支持
- ⚠️ 不确定，需搜索验证
- ❌ 已知不支持
- 🔶 可通过替代方案实现

## 一、玩法类型支持

| 玩法类型 | 状态 | 备注 |
|---------|------|------|
| 冒险闯关 | ✅ | 基础类型 |
| 跑酷 | ✅ | 有官方示例 |
| 解谜 | ✅ | 节点图触发器实现 |
| 对抗竞技 | ✅ | 多人系统支持 |
| 模拟经营 | ✅ | 需节点图模拟资源系统 |
| 多人派对 | ✅ | 大厅多人房间 |
| 塔防 | 🔶 | 需节点图模拟寻路和波次 |
| 回合制战斗 | 🔶 | 需状态暂停+指令序列 |
| 视觉小说 | 🔶 | 节点图模拟对话推进 |
| RTS | ❌ | 无鼠标框选控制 |
| MOBA | ✅ | 有社区实机案例（BV1T3vyB4EJN），含ADC/打野/红BUFF等完整MOBA机制 |
| 射击 | ✅ | 有社区教程「枪械精通」（BV1aJyGBPEho），含完整射击系统实现 |
