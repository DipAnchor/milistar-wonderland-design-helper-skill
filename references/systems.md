# 千星奇域 · 系统分类参考

> 本文件按类别列出千星奇域的主要系统，帮助快速定位问题涉及的领域。

## 一、角色系统

| 系统 | 说明 |
|------|------|
| 奇偶 | 双角色（男/女），千星奇域专属角色 |
| 装扮 | 发型/上装/下装/头饰/背饰/耳饰等 |
| 妆容 | 眼瞳、面妆等面部定制 |
| 套装效果 | 整套装扮激活特殊效果 |
| 搭配方案 | 多套装方案，战斗中随机切换 |
| 部件调整 | 位置/大小/角度/隐藏部件 |

## 二、编辑器系统（千星沙箱）

| 系统 | 说明 |
|------|------|
| 物件库 | 内置场景物件、机关、装饰 |
| 节点图编辑器 | 可视化逻辑编程（事件→条件→行动） |
| 触发器 | 区域/时间/条件触发 |
| 变量系统 | 数值/布尔/字符串变量管理 |
| 多人配置 | 团队分配、同步、计分 |
| 成就定义 | 自定义成就条件和奖励 |
| 测试功能 | 内置关卡测试 |

## 三、关卡系统

| 系统 | 说明 |
|------|------|
| 典藏模板 | 官方提供的关卡模板，可自己创建 |
| 自定义模板 | 基于典藏模板编辑保存 |
| 大厅 | 个人/好友/公开房间 |
| 排行榜 | 竞速/评分排名 |
| 段位 | 竞技积分段位（新星0星等） |
| 赛季 | 准备期→游玩期→展示期 |

## 四、经济系统

| 系统 | 说明 |
|------|------|
| 颂愿（常驻） | 使用重现晶簇，4星套装形录 |
| 颂愿（活动） | 使用绮刻之楔/明刻之楔，5星+4星套装 |
| 纪游 | 天空/诸界(¥38)/诸界史诗(¥98) |
| 千星商城 | 使用事象凝核购买装扮 |
| 重现晶簇 | 活跃获得，用于常驻颂愿 |
| 事象凝核 | 商城货币 |

## 五、成长系统

| 系统 | 层级 | 说明 |
|------|------|------|
| 奇域等级 | 2~20级 | 游玩关卡获得经验，每周上限 |
| 奇匠等级 | Lv.1~5 | 创作关卡提升，解锁编辑功能 |
| 探索星章 | 20个条件 | 完成探索任务获得奖励 |
| 绮衣收藏家 | 随缤纷评分提升 | 收集装扮的等级展示 |

## 六、社交系统

| 系统 | 说明 |
|------|------|
| 个人大厅 | 3小时持续，可邀请好友 |
| 好友大厅 | 访问好友的大厅 |
| 装扮赠送 | 奇域10级/冒险30级+默契3级/好友14天 |
| 装扮分享码 | 生成装扮分享给他人 |
| 多人匹配 | 时尚度排名展示 |
